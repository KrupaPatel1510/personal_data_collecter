# Interactive Personal Data Collector

## 📌 Project Description
The **Interactive Personal Data Collector** is a simple Python program that collects basic personal information from the user, processes it, and displays the results. It demonstrates the use of fundamental Python concepts such as variables, data types, user input, type casting, arithmetic operators, and built-in functions.

## 🚀 Features
- Accepts user input for:
  - Name
  - Age
  - Height (in cm)
  - Favorite number
- Calculates the approximate birth year.
- Displays all collected information.
- Shows the data type and memory address of each variable using `type()` and `id()`.

## 🛠️ Technologies Used
- Python 3

## 📂 Program Concepts Covered
- `print()` function
- `input()` function
- Variables
- Data Types
- Type Casting (`int()`, `float()`)
- Arithmetic Operators
- Built-in Functions:
  - `type()`
  - `id()`

## ▶️ How to Run
1. Make sure Python 3 is installed on your computer.
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the program using:

```bash
python filename.py
```

Replace `filename.py` with the name of your Python file.
